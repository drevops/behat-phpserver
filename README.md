## Features

Behat Context to enable PHP server for tests.

## Installation

    composer require --dev drevops/behat-phpserver

## Usage

    cd ./tests/behat/ && ../../vendor/bin/behat --colors

## Maintenance

### Lint code

```bash
composer lint
composer lint:fix
```

### Run tests

```bash
composer test
```
